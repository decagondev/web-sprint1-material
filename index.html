First Steps into JavaScript

Developers work with values, variables, and logic daily, which you'll learn first. These building blocks will inform the rest of your career as a web developer.

Values

Learn about JavaScript's primitive value types and how they form the foundation of programming.

Variables

Store and access values using variables, making your code more flexible and reusable.

Comparison and Logic

Compare values and build logical expressions to control your program's flow.

Using Dev Tools

Learn to use browser development tools to debug and analyze your JavaScript code.

Program Flow

A good developer will code their program to do different things based on variables. We call these conditional statements, and they're another foundational tool for building incredible programs.

Conditional Code

Create different paths in your code using conditional statements to respond to different scenarios.

Falsiness

Understand how JavaScript evaluates different values as truthy or falsy in conditional contexts.

Advanced Logic

Combine multiple conditions using logical operators to create complex decision-making structures.

Module Project: Program Flow

Apply conditional logic to create dynamic, responsive programs that react to different inputs.

Functions

Functions allow you to assign a task to a section of code and then reuse that code by typing the task name. If that sounds like magic, it is. You'll learn how to create and call a function and how functions can return new values to you.

Declaring a Function

Learn to write functions that package code into reusable, named blocks.

Passing Arguments

Make functions more flexible by allowing them to receive data to work with.

Returning Values

Learn how functions can produce new values that can be used elsewhere in your code.

Module Project: Functions

Create reusable code with functions that can accept input and return output in a real-world application.

HTML & CSS I

Even the most complex web applications start with simple HTML elements. Knowing how to use (and style!) HTML has a massive impact on your future as a developer. This module will explore the fundamentals of HTML.

The Anatomy of an HTML Document

Learn the structure and elements that make up an HTML document.

Using HTML Tag Attributes

Add information and functionality to HTML elements with attributes.

Using the Style Tag

Apply CSS to your HTML to control the appearance of your web pages.

Including the Script Tag

Add JavaScript to your HTML pages to create interactive websites.
