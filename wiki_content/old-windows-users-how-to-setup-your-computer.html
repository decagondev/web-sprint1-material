Let's set up your computer like a professional!

You'll install any necessary software and configure your settings so that you can be successful with future projects in your BloomTech learning journey. This will redefine how you interact with your computer. Below you'll find instructions on how to set up your machine and test that everything has been set up correctly. You may encounter software, terminology, and processes that are new to you. If it feels a little overwhelming, don't worry! We've all been there and you can do this!

Here's some advice to help you complete this part of the Orientation:

  • Take your time - When configuring our work areas, it is better to be methodical and patient. Make sure you set aside sufficient time and have a quiet, distraction free environment.
  • Go in order - It may be tempting to jump around or skip steps, but often these requirements will build upon each other. Make sure you are confident with a set up stage before progressing to the next one.
  • Settings matter - If you already have some or all of these configurations on your computer, please watch the videos to double check that your settings match ours.
  • Remember help is available! If you encounter issues while going through this process, there will be a link to schedule a one-on-one help session with a learning assistant.

Please, watch the videos in order, and do not skip any. Besides installation, proper settings for each tool is discussed.
